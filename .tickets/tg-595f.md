# Write minimal busybox config

Create busybox.config with minimal applets for networking:
- Shell (ash)
- Init system
- Networking: ip, udhcpc, ping, wget (as backup to curl)
- File utilities: ls, cat, cp, mv, rm, mkdir, chmod
- Text: echo, printf, sed, grep
- System: mount, umount, ps, kill, reboot

Disable everything else to minimize size.

Reference: docs/image-build-plan-1769311563.md

## Acceptance Criteria

- Busybox builds with config
- All required networking tools present
- Size <= 600KB
