# Build buildroot images

Run images/buildroot/build.sh to compile the minimal Linux image. This downloads Buildroot, cross-compiles the kernel and rootfs with musl, curl, and OpenSSL. Expected outputs: images/output/kernel-riscv64.bin and images/output/root-minimal.bin

## Acceptance Criteria

- build.sh completes successfully
- images/output/kernel-riscv64.bin exists
- images/output/root-minimal.bin exists
