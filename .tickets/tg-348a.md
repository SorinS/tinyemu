# Write minimal kernel config for TinyEMU

Create linux.config fragment with:
- Size optimization (CONFIG_CC_OPTIMIZE_FOR_SIZE)
- VirtIO drivers (PCI, BLK, NET, CONSOLE)
- HVC console for TinyEMU
- Basic networking (no IP_PNP, userspace DHCP)
- ext4, proc, sys, tmpfs, devtmpfs
- Disable: sound, USB, wireless, bluetooth, DRM, framebuffer, NFS, SCSI, debug

Target kernel size: ~2.5MB

Reference: docs/image-build-plan-1769311563.md

## Acceptance Criteria

- Kernel boots in TinyEMU
- VirtIO block and network devices work
- Console output visible
- Kernel size <= 3MB
