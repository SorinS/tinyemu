# Improve test coverage for 9P filesystem and VirtIO-9P device.

## Background
- p9/ package: 56.6% coverage (target: 75%+)
- virtio/p9.go handlers: 0% coverage

9P is used for host filesystem sharing. While not critical for basic boot, it's a significant feature that's undertested.

## Tests Needed

### p9/ package
- Message parsing/marshaling for all message types
- hostfs operations (Remove, Setattr, Mknod, Link, Lock, etc.)
- Error handling paths

### virtio/p9.go
- recvRequest handler
- Protocol handlers: handleVersion, handleAttach, handleWalk, etc.
- Error response generation

## Acceptance Criteria
- p9/ coverage reaches 75%+
- virtio/p9.go key handlers are tested
- Integration test: mount 9P share, read/write files
