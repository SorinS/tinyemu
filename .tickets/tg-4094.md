Create a minimal Ubuntu image using prebuilt ubuntu-base tarball that boots in TinyEMU with working networking (DHCP + DNS) and curl for HTTPS requests. Uses the shared kernel from the Buildroot build. Target size: <40MB total.

## Acceptance Criteria

- Image boots successfully in TinyEMU with Buildroot kernel
- DHCP acquires IP from slirp (10.0.2.15)
- DNS resolution works
- curl https://example.com succeeds
- Total size (kernel + rootfs + BBL) < 40MB
