# Write Ubuntu image build script

Create images/ubuntu/build.sh that:
1. Downloads ubuntu-base-24.04-base-riscv64.tar.gz if not present
2. Extracts to work/rootfs/
3. Uses QEMU user-mode emulation for chroot (qemu-user-static)
4. Configures inside chroot:
   - netplan for DHCP on eth0
   - serial console on hvc0
   - root password
   - /etc/fstab
5. Installs curl and ca-certificates via apt
6. Cleans up to reduce size (apt cache, docs, locales, logs)
7. Creates ext4 image with mkfs.ext4 -d
8. Copies to images/output/root-ubuntu.bin

Reference: docs/image-build-plan-1769311563.md

## Acceptance Criteria

- Script handles QEMU user-mode setup
- Build completes without errors
- Output image created at images/output/root-ubuntu.bin
