# Implement small packet escape char ACK optimization

Please see docs/1768939013-tcp-input-full-implementation-plan.md for context.

Implement the small packet optimization that immediately ACKs packets starting with escape character.

C Reference: tcp_input.c:1243-1246

Code:
if (ti->ti_len && (unsigned)ti->ti_len <= 5 &&
    ((struct tcpiphdr_2 *)ti)->first_char == (char)27) {
    tp->t_flags |= TF_ACKNOW;
}

This optimization is for terminal applications - small packets (1-5 bytes) starting with ESC (0x1B) get immediate ACK to improve interactive response time.

Implementation:
1. After data processing, check segment length is 1-5
2. Check first byte of data is 0x1B (escape)
3. If both true, set TF_ACKNOW
