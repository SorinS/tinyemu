# Implement ICMP echo request forwarding to external hosts

In ip_icmp.go:195-199, ICMP echo requests to non-vhost addresses are silently dropped. The C slirp code would create a socket and send a real ping to the external host and forward the reply back. This affects guest ping to internet hosts.
