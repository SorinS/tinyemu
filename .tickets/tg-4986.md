# Verify buildroot image size < 10MB

Verify total image size meets target:
1. Check kernel size: ls -lh images/output/kernel-riscv64.bin
2. Check rootfs size: ls -lh images/output/root-minimal.bin  
3. BBL size: ls -lh testdata/boot/bbl64.bin
4. Calculate total
5. If over budget, identify largest components and optimize

Target breakdown:
- Kernel: ~2.5MB
- RootFS: ~7MB
- BBL: ~50KB
- Total: <10MB

Reference: docs/image-build-plan-1769311563.md

## Acceptance Criteria

- Total size < 10MB
- Document actual sizes in ticket notes
