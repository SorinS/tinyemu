# tinyemu-go

Hello humans, fellow human here.

This repo contains a Claude LLM generated transliteration of Fabrice Bellard's
[TinyEMU](https://bellard.org/tinyemu/) RISC-V emulator from C to Go.

Perhaps the interesting entry point is [temubox/example/main.go](https://github.com/jtolio/tinyemu-go/blob/main/temubox/example/main.go).

[I wrote a blog post describing the genesis of this project.](https://www.jtolio.com/2026/01/a-pure-go-linux-environment-written-by-claude-directed-by-fabrice-bellard/)
